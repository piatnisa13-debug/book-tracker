# Трекер прочитанных книг

Консольное приложение на Python для учёта прочитанных книг.
Данные хранятся в файле books.json.

## Возможности

- Добавление книги (автор, название, оценка от 1 до 5, дата прочтения)
- Проверка на дубликаты при добавлении
- Просмотр списка всех книг
- Подсчёт средней оценки
- Статистика по авторам (сколько книг у каждого)
- Удаление книги по номеру

## Как запустить

1. Установите Python 3
2. Запустите программу командой:
python main.py
